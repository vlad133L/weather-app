## Available Scripts

In the project directory, you can run:

### `npm run start`

Runs the app.\
Open [ http://192.168.100.13:8080](http://192.168.100.13:8080) or [http://127.0.0.1:8080](http://127.0.0.1:8080) to view it in your browser.

### `npm run server`

Runs the server.

### `npm run client`

Runs the client.
